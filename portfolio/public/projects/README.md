# Project Images Directory

Each project should have its own folder with the following images:

## Folder Structure

```
projects/
├── general-mills/
│   ├── hero.jpg         (Main project image, 16:9 ratio, ~1920x1080px)
│   └── thumbnail.jpg    (Grid thumbnail, 16:10 ratio, ~1200x750px)
├── sourcewell/
│   ├── hero.jpg
│   └── thumbnail.jpg
├── boston-scientific/
│   ├── hero.jpg
│   └── thumbnail.jpg
└── ... (etc.)
```

## Image Specifications

### Hero Image
- **Aspect Ratio**: 16:9
- **Recommended Size**: 1920x1080px
- **Max File Size**: 500KB
- **Format**: JPG (preferred) or PNG
- **Usage**: Project detail page header

### Thumbnail Image
- **Aspect Ratio**: 16:10
- **Recommended Size**: 1200x750px
- **Max File Size**: 300KB
- **Format**: JPG (preferred) or PNG
- **Usage**: Project grid cards, carousel

## Project Slugs (Folder Names)

Based on your projects, create folders with these exact names:
- `general-mills`
- `sourcewell`
- `boston-scientific`
- `ice`
- `southwest`
- `mde`
- `nebraska`

## Tips

- Use descriptive, high-quality screenshots of your work
- Blur or anonymize sensitive client information if needed
- Optimize images for web (use TinyPNG, ImageOptim, or similar)
- Maintain consistent visual style across all project images
- Consider adding branded overlays or frames for a polished look
